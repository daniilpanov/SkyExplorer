Игра будет называться "Космические Приключения" || "Star Adventures"
_______________________________________________
Необходимые классы: 
 - Main (главный класс; наследует JFrame) : стандартный класс
 - Menu (класс для отображения и управления меню; наследует JWindow; объект создаётся в классе Main) : стандартный класс
 - Sprite (класс для манипуляции спрайтом) : универсальный стандартный класс
 - Img (статический класс, хранящий объекты типа java.awt.Image) : стандартный класс
 - Stone (класс для управления метеоритами) :
	+ генерация метеоритов (статический; возвращает объект типа Runnable)
	+ расчёт траектории метеорита (возвращает массив, состоящий из двух чисел - отклонения по x & y за один проход таймера)
	+ анимация метеорита (void; координаты увеличиваются на значения в массиве + поворот метеорита + проверка на столкновение)
 - StarShip (класс для управления кораблём) : 
	+ слушатель клавиатуры (управление - движения и стрельба)
	+ полёт снаряда (расчёт траектории, анимация)
 - Game (класс для отображения и управления игрой; наследует JPanel) : 
	+ отображение всех спрайтов
	+ отображение фона
	+ игровое меню (показывается во время игры)
